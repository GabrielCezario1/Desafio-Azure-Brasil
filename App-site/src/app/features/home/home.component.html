Repositório Base para Projetos Angular

Um ponto de partida robusto para novos projetos, com uma estrutura organizada e funcionalidades essenciais.

Sobre o Projeto

Este repositório foi criado para servir como uma base sólida e organizada para iniciar novas aplicações com Angular. Ele já vem com uma estrutura de pastas bem definida (`core`, `features`, `shared`), facilitando a manutenibilidade e a escalabilidade do código.

A base inclui funcionalidades prontas para uso, como um sistema de gerenciamento de usuários — implementado com Formulários Reativos (Reactive Forms) para maior controle e testabilidade — e um componente de paginação reutilizável, permitindo que você foque em construir as funcionalidades específicas do seu projeto.

Tecnologias Utilizadas

Angular

Framework principal para a construção da interface.

Bootstrap

Para componentes de UI e layout responsivo.

SCSS

Pré-processador de CSS para estilos avançados.
